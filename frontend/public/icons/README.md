# PWA Icons

Icons duoc generate tu `icon-master.svg` va `icon-maskable-master.svg` qua `npm run generate:icons`. Chay lai script khi doi brand.

## Kich thuoc duoc tao ra

| File                                      | Kich thuoc | Purpose    |
| ----------------------------------------- | ---------- | ---------- |
| `icon-192.png`                            | 192x192    | any        |
| `icon-512.png`                            | 512x512    | any        |
| `icon-maskable-512.png`                   | 512x512    | maskable   |
| `apple-touch-icon.png`                    | 180x180    | iOS        |
| `../favicon-32.png`                       | 32x32      | favicon    |
| `../og-default.png`                       | 1200x630   | OG social  |

Icon `maskable` co safe zone 80% (noi dung chinh nam trong hinh tron noi tiep de khong bi launcher crop khi hien thi shape adaptive).
